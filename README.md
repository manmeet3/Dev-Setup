# Minimal Developer Terminal Stack

## Terminal
- Ghostty — GPU-accelerated terminal emulator

## Shell
- Zsh — interactive shell

## Prompt
- Starship — fast cross-shell prompt

## Terminal Multiplexing
- tmux — persistent terminal sessions and pane splitting

## Editor
- Neovim — modern Vim-based editor

## Neovim Configuration
- LazyVim — preconfigured Neovim environment

## Core CLI Tools
- git — version control
- curl — HTTP downloads / API requests

## Productivity CLI Tools
- fzf — fuzzy finder
- ripgrep (`rg`) — fast code search
- eza — modern `ls` replacement
- bat (or `batcat` on Linux) — modern `cat` replacement

## Linux-only Support Tools
- build-essential — compilers and build tools
- unzip — archive extraction
- xclip — clipboard integration

## Package Managers
- Homebrew (macOS)
- apt (Linux)

## Config Files Created
- ~/.zshrc
- ~/.config/starship.toml
- ~/.tmux.conf
- ~/.config/ghostty/config.ghostty
- ~/.config/nvim
